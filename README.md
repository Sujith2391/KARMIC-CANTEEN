# Karmic Canteen

A React-based application for managing meal selections, employee dashboards, and administrative functions for Karmic Canteen.

## Live Website

The application is deployed and running at: [https://karmic-canteeen.netlify.app/](https://karmic-canteeen.netlify.app/)

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```
   npm install
   ```

2. Run the development server:
   ```
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:5173` (or the port shown in the terminal).

## Build for Production

To build the app for production:

```
npm run build
```

Then, preview the build:

```
npm run preview
```

## Technologies Used

- React
- TypeScript
- Vite
- Supabase (for backend services)
